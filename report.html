2.1. Dueling Network¶

Introducing Deuling Network. Look at the picture below. The right side of the figure is A(s, ) = Q(s, ) - V(s), which is advantage funtion.

(source from torch.ch)

V (s) and A (s, a) are arranged below the output layer of Q (s,). It is possible to learn V (S) which is determined only by the state without relying on the actions, thereby improving the training performance.

I have improved the DQN neural network as below.

For instance, Q(s, a1) = V(s) + Adv(s,a1) - (Adv(s,a1) + Adv(s,a2) + Adv(s,a3) + Adv(s,a4) / 4

2.2. DDQN¶

DQN often overestimates the maximum-value of Q value since DQN uses max_Q(S, A) in the target. A technique designed to reduce this overestimation is known as DDQN.

In the next code cell, I wrote the above formula in code. There are two networks used for training. In the Agent Class, these are the qnetwork_target and qnetwork_local . In the next state, the maximum action is obtained from qnetwork_local, and the Q-value at this time-step is come from qnetwork_target.

# Get action maximizing Q values from local network
max_actions = self.qnetwork_local(next_states).max(1)[1].unsqueeze(1)
        
# Get Q values from target network for next state
TargetQnext = self.qnetwork_target(next_states).detach().gather(1, max_actions)
        
# Compute Q values as targets
TargetQ = rewards + (gamma * TargetQnext * (1 - dones))

# Get expected Q values from local_network
ExpectedQ = self.qnetwork_local(states).gather(1, actions)

# Compute loss
Loss = F.mse_loss(ExpectedQ, TargetQ)

2.3. Experience Replay¶

Experience Replay is to store in replay memory the experiences that agents gain from exploring in the environment. As the agent learns, it randomly takes several samples from this memory and updates the neural network. So,it can eliminate the correlation between samples. If a SARSA agent is trapped between a green banana and a wall, it will be trained for that situation, and it will never escape from bad situation.

We can solve this problem by using Experience Replay. This is because the samples used for learning do not have a time correlation like On-policy algorithm. In addition, the training is stable because the neural network is updated through several samples extracted from replay memory. In this project, I used the proven ReplayBuffer class.
